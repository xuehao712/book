# 定期付款

自动付款要求用户提前与商家签署付款协议\(例如，第三方预扣协议\)。 之后，当用户在商家站点上购买商品时，商家可以使用授权令牌直接从用户帐户中扣除款项。

要使用此付款解决方案，商家需要提前获得授权。 用户需要与商家签订合同\(例如，第三方预扣协议\)以授权商家获得授权令牌。 之后，商家可以使用令牌发起自动借记付款请求。
