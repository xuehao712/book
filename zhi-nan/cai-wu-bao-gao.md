# 财务报告

Yuansfer支持各种类型的财务报告，您可以在我们的网站中过滤并获取符合您要求的报告。

### 报告类型

您可以按供应商类型获取报告。

#### 商家报告

| **列** | **词义** |
| :--- | :--- |
| Date | 报告日期 |
| Merchant Account | 商户信息 |
| Transaction Amount | 交易金额 |
| Currency | 交易的结算货币 |

#### 店铺报告

| **列** | **词义** |
| :--- | :--- |
| Store ID | 商店的唯一ID |
| Date | 报告日期 |
| Payments | 已完成交易的总金额 |
| Refunds | 退款交易总额 |
| Total Gross | 商店获得的总金额 |

### 过滤数据

#### 根据日期

您可以根据报告日期过滤数据。

#### 根据支付方式

您可以根据支付方式过滤数据。

#### 导出报告

您可以在需要时将报告导出为Excel工作表。 您可以在门户网站的右上方找到“下载”按钮。
